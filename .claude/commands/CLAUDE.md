# Claude Code slash commands

Custom commands live in .claude/commands/ as .md files.
Invoke with /command-name in any Claude Code session.

## Available commands (create these files)
- /new-migration — scaffold a new Supabase migration file with header comment
- /new-service — scaffold a new service + repository interface + Zod schema
- /new-component — scaffold a typed React component with JSDoc prop interface
- /check-adr — list all ADRs in docs/adr/ and their status
- /sprint-status — show current sprint checklist from docs/sprint-current.md
