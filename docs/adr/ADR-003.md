# ADR-003 — RLS como fuente de verdad de permisos

**Status:** Accepted  
**Sprint:** 0

## Decision
All permission enforcement lives in Supabase RLS policies. UI restrictions are UX only, not security.

## Reason
Frontend can be bypassed. Security must live at the data layer.

## Rejected alternatives
- Middleware-only checks: bypassable via direct API calls
