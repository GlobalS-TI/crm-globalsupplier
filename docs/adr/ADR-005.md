# ADR-005 — No AWS en Fase 1

**Status:** Accepted  
**Sprint:** 0

## Decision
Supabase + Vercel for all infra in Phase 1.

## Reason
7 users, ~500 records. AWS adds operational complexity a single developer would have to maintain alone.

## Revisit trigger
> 50 active users, or compliance requirements (HIPAA, etc.)
