# ADR-002 — Campo stale en opportunities

**Status:** Accepted  
**Sprint:** 1

## Decision
stale is a stored boolean column updated by a Postgres trigger when last_activity_at exceeds 7 days.

## Reason
Avoids computing the flag on every dashboard query. Indexable. Dashboard reads it directly.

## Rejected alternatives
- Compute in service layer on each request: O(n) on every dashboard load, not indexable
